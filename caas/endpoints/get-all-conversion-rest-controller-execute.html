List user's conversions.

GET 
/conversions
deprecated

This endpoint has been deprecated and may be replaced or removed in future versions of the API.

Get a list of user's conversions.

Request​

Query Parameters

    page number

    Default value: 1

    Page number.

    size number

    Possible values: <= 100

    Default value: 20

    Page size. Max size is 100.

    sort string

    Possible values: [created_at]

    Page sort attribute.

    order string

    Possible values: [asc, desc]

    Default value: asc

    Page order.

    operation_id string

    Operation ID for conversion.

    currency_symbol string

    Currency Symbol for conversion.

    quotation_id string

    Quotation ID for conversion.

    conversion_type string

    Possible values: [buy, sell]

    Conversion type.

    client_name string

    Conversion client name.

    client_document string

    Conversion client document.

    created_at_start date-time

    Created at start for any conversion.

    created_at_end date-time

    Created at end for any conversion.

Header Parameters

    nonce stringrequired

    The nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier.

Responses​

  • 200
  • 400
  • 401
  • 422

The conversions returned successfully.
