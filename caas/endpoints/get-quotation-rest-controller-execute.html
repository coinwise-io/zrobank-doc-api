Get a quotation.

GET 
/quotations
deprecated

This endpoint has been deprecated and may be replaced or removed in future versions of the API.

Get currency quotation. Must use the path "quotations/spot".

Request​

Query Parameters

    base_currency stringrequired

    Conversion currency.

    Example: BTC
    amount_currency stringrequired

    Conversion currency quote.

    Example: BRL
    amount numberrequired

    Conversion amount in cents.

    Example: 1
    side stringrequired

    Possible values: [buy, sell]

    Conversion side.

    Example: buy

Header Parameters

    nonce stringrequired

    The nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier.

Responses​

  • 200
  • 400

Quotation returned successfully.

  • application/json
