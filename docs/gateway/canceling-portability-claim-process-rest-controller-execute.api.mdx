<MethodEndpoint method={"delete"} path={"/pix/keys/{id}/portabilities/cancel"}></MethodEndpoint>



Cancel portability process. Portability will be stopped and the key will be canceled. If key state is:
    <ul>
        <li>PORTABILITY_STARTED: portability process will be canceled and key state will change to PORTABILITY_CANCELING.
    </ul>
    Return key which state is PORTABILITY_CANCELING.
