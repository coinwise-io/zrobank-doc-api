<h1 className={"openapi__heading"}>Cancel pix key.</h1>

<MethodEndpoint method={"delete"} path={"/pix/keys/{id}/code"}></MethodEndpoint>



Cancel pix key process. This route works to PENDING or CLAIM_PENDING. If key state is:
    <ul>
      <li>PENDING: key state will change to CANCELED.
      <li>CLAIM_PENDING: a message will be sent to DICT and the key will be released to the claimer. Key state will change to CLAIM_CLOSING.
    </ul>
    Return key which state is CANCELED or CLAIM_CLOSING.
