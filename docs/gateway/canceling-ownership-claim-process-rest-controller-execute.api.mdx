<h1 className={"openapi__heading"}>Cancel a ownership claim that was created by this ZRO user.</h1>

<MethodEndpoint method={"delete"} path={"/pix/keys/{id}/claims/cancel"}></MethodEndpoint>



Cancel ownership process. Portability will be stopped and the key will be canceled. If key state is:
    <ul>
        <li>OWNERSHIP_WAITING: ownership process will be canceled and key state will change to OWNERSHIP_CANCELING.
    </ul>
    Return key which state is OWNERSHIP_CANCELING.
