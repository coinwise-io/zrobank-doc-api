<h1 className={"openapi__heading"}>Cancel portability process.</h1>

<MethodEndpoint method={"delete"} path={"/pix/keys/{id}/portabilities/approve"}></MethodEndpoint>



Cancel portability process. Portability will be stopped and the key will remain in Zrobank. If key state is:
    <ul>
        <li>PORTABILITY_REQUEST_PENDING: portability process will no be approved and key state will change to READY.
    </ul>
    Return key which state is READY.
