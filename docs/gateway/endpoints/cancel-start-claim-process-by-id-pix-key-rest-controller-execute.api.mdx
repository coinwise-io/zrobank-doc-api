<h1 className={"openapi__heading"}>Cancel ownership claim start process.</h1>

<MethodEndpoint method={"delete"} path={"/pix/keys/{id}/claims/start"}></MethodEndpoint>



Cancel ownership start process. Ownership claim process will not be started and key will remain in third party ISPB.
    If key state is:
    <ul>
      <li>OWNERSHIP_PENDING: ownership claim process will not start and key state will change to CANCELED.
    </ul>
    Return key which state is CANCELED.
