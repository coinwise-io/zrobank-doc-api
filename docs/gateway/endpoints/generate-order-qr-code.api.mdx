<h1 className={"openapi__heading"}>Generate new dynamic QR Code for single or multiple orders</h1>

<MethodEndpoint method={"post"} path={"/api/transaction/generate_order_qr_code_pix"}></MethodEndpoint>



Generates a new dynamic QR code for an individual order or for multiple orders at once.
    The orders must be specified in an array of objects. For a single order, the array will be composed by one single object.
    For multiple orders, the array will be composed by multiple objects. Shipping fee: When the orders have shipping fee, you must
    specify it in another order in the array, but for this order, the 'fee' must be 0, the 'company_identifier' should be the requesting
    company's default wallet id and the 'value' should be the shipping fee value.
