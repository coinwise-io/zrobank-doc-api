<h1 className={"openapi__heading"}>Cancel portability start process.</h1>

<MethodEndpoint method={"delete"} path={"/pix/keys/{id}/portabilities/start"}></MethodEndpoint>



Cancel portability start process. Portability process will not be started and key will remain in third party ISPB.
    If key state is:
    <ul>
      <li>PORTABILITY_PENDING: portability process will not start and key state will change to CANCELED.
    </ul>
    Return key which state is CANCELED.
