<h1 className={"openapi__heading"}>Approve portability start process.</h1>

<MethodEndpoint method={"post"} path={"/pix/keys/{id}/portabilities/start"}></MethodEndpoint>



Allow Zrobank to start a portability process. If key state is:
      <ul>
        <li>PORTABILITY_PENDING: portability start process will be sent to DICT.
          After that, the user should be asked to approve portability in third party ISPB. Key state will change to PORTABILITY_OPENED.
      </ul>
      Return key which state is PORTABILITY_OPENED.
