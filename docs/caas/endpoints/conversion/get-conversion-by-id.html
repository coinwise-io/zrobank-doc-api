Get conversion by ID

GET/otc/conversions/{id}

Enter the conversion's ID and execute to get all it's information

Request Try it!​

Header​

TitleTypePropertiesMaximum LengthDescription
nonce🔸STRINGMust be a UUID-The nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier

🔸 Required parameters on header

Parameters​

TitleTypePropertiesDescription
id🔸STRINGMust be a UUIDConversion ID

🔸 Required parameters on path
