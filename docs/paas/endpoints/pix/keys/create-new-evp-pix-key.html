Create new EVP PIX key

POST/pix/keys

Create a new EVP key type (random key) to the user

Request Try it!​

Header​

TitleTypePropertiesMaximum LengthDescription
x-transaction-uuid🔸STRINGMust be a UUID-The transaction ID is a UUID (v4) used to uniquely identify the object that will be created. All requisitions must have an identifier
nonce🔸STRINGMust be a UUID-The nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier
