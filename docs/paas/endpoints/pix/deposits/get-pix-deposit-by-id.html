Get PIX deposit by ID

GET/v2/pix/deposits/{id}

Enter the PIX deposit's ID below and execute to get it's state and all information.

Request Try it!​

Header​

TitleTypePropertiesDescription
x-wallet-uuidSTRINGMust be a UUIDSender Wallet UUID (if empty, your default Wallet UUID will be settled)
nonce🔸STRINGMust be a UUIDThe nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier

🔸 Required parameters on header

Parameters​

TitleTypePropertiesDescription
id🔸STRINGMust be a UUIDDeposit ID

🔸 Required parameters on path
