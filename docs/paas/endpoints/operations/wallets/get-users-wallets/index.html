List user's wallets

GET/operations/wallets

Get a list of user's wallets. You can include any of the filter parameters below to refine your search.

Request Try it!​

Header​

TitleTypePropertiesDescription
nonce🔸STRINGMust be a UUID(v4)The nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier

🔸 Required parameters on header

Filter Parameters​

TitleTypePropertiesDescription
ownerSTRINGAvailable values:
USER, OTHER		User wallet owner type
