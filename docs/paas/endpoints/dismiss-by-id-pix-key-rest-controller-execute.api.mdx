<h1 className={"openapi__heading"}>Dismiss pix key by ID.</h1>

<MethodEndpoint method={"post"} path={"/pix/keys/{id}/dismiss"}></MethodEndpoint>



Set key to do not be shown anymore. The key state will be changed to CLAIM_PENDING, CANCELED or READY. If key state is:
    <ul>
      <li>CLAIM_NOT_CONFIRMED: key state will change to CLAIM_PENDING.
      <li>PORTABILITY_REQUEST_AUTO_CONFIRMED: key state will change to CANCELED.
      <li>PORTABILITY_CANCELED: key state will change to CANCELED.
      <li>OWNERSHIP_CANCELED: key state will change to CANCELED.
      <li>OWNERSHIP_CONFLICT: key state will change to CANCELED.
      <li>PORTABILITY_READY: key state will change to READY.
      <li>OWNERSHIP_READY: key state will change to READY.
      <li>ADD_KEY_READY: key state will change to READY.
      <li>DELETED: key state will change to CANCELED.
      <li>NOT_CONFIRMED: key state will change to CANCELED.
    </ul>
    Returns user's pix key which state is CLAIM_PENDING, CANCELED or READY.
