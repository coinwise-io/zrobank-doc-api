Get receipt by its operation id.

GET 
/pix/receipt/by-operation/:id
deprecated

This endpoint has been deprecated and may be replaced or removed in future versions of the API.

Enter the pix payment's operation ID below and execute to get its receipt. Should use /operations/{id}/receipt path.

Request​

Path Parameters

    id stringrequired

    Operation id.

Header Parameters

    x-wallet-uuid string

    Sender Wallet UUID (if empty, your default Wallet UUID will be settled)

    nonce stringrequired

    The nonce ID is a UUID (v4) used to uniquely identify the requisition. All requisitions must have an identifier.

Responses​

  • 200
  • 400
  • 401
  • 422

Receipt found by operation id successfully.
